## вңЁ Features

| Feature | Description |
|---------|-------------|
| рҹҢҫ **AI Crop Advisory** | Personalized crop recommendations via Claude AI based on soil, region & season |
| рҹҢҰпёҸ **Weather Forecasting** | Real-time hyperlocal weather data to plan farming activities smartly |
| рҹ“Ҡ **Live Mandi Rates** | Live market crop prices вҖ” help farmers get fair rates |
| рҹӨ– **AI Chatbot (Hindi)** | Voice & text chatbot powered by Claude AI вҖ” supports Hindi language |
| рҹҰ  **Disease Detection** | Upload a crop image вҶ’ instant AI-based disease diagnosis & remedy |
| рҹ‘Ҙ **Community Forum** | Groups & messaging for farmers to connect and share knowledge |
| рҹ”җ **Secure Auth** | JWT-based farmer registration & login system |
| рҹ“ў **Smart Alerts** | Real-time weather & farming alerts with audio notification support |

### рҹ“Ұ Step-by-Step Setup

#### 1пёҸвғЈ Clone the Repository

```bash
git clone https://github.com/coder-Yash886/farmvaani-agritech.git
cd farmvaani-agritech/farmvaani-app
```

---

#### 2пёҸвғЈ Setup & Run the Backend

```bash
cd Backend
npm install
```
Create a `.env` file inside `Backend/` (see [Environment Variables](#-environment-variables) section):

```bash
# Start backend in development mode
npm run dev
```

> вң… Backend runs at: `http://localhost:5000`

---

#### 3пёҸвғЈ Setup & Run the Frontend

```bash
cd ../Frontend
npm install
```

Create a `.env` file inside `Frontend/`:

```bash
# Start frontend dev server
npm run dev
```

> вң… Frontend runs at: `http://localhost:5173`

---

#### 4пёҸвғЈ Open in Browser

```
http://localhost:5173
```

## рҹ”җ Environment Variables

### Backend вҖ” `Backend/.env`

```env
PORT=5000
MONGO_URI=mongodb+srv://<username>:<password>@cluster.mongodb.net/farmvaani
JWT_SECRET=your_super_secret_jwt_key
GROQ_API_KEY=your_groq_api_key
WEATHER_API_KEY=your_openweather_or_weatherapi_key
```

### Frontend вҖ” `Frontend/.env`

```env
VITE_API_BASE_URL=http://localhost:5000/api
```

| Variable | Description | Required |
|----------|-------------|:--------:|
| `PORT` | Backend server port | вң… |
| `MONGO_URI` | MongoDB Atlas connection URI | вң… |
| `JWT_SECRET` | Secret key for signing JWT tokens | вң… |
| `GROQ_API_KEY` | Groq LLM API key | вң… |
| `WEATHER_API_KEY` | Weather data API key | вң… |
| `VITE_API_BASE_URL` | Backend base URL for frontend | вң… |

|--------|-------|-------------|------|
